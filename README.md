## Environment variables

This project uses a local `.env` file for sensitive configuration such as Supabase credentials.

1. Copy `.env.example` to `.env`.
2. Fill in your real values in `.env`.
3. Run the project normally; the generated file `lib/core/constants/env.g.dart` will use those values.

Important:

- `.env` should never be committed to the repository.
- `.env.example` is the safe template to share with other developers.
